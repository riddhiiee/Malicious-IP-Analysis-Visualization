# 🔍 Malicious IP Address Analysis

A project to collect, analyze, and visualize malicious IP data to uncover patterns in abuse types and regions.

---
 
## 🚀 Features
- Scraped 2000+ malicious IP addresses along with their abuse reasons
- Categorized data into 5 major abuse types (Spam, Hijacking, etc.)
- Cleaned and preprocessed raw data for consistency
- Visualizes insights in Tableau (country, type, IPv4 vs IPv6)
- Built interactive dashboards to visualize global trends
---

## 🛠️ Tech Stack

- **Python (Selenium)** – Web scraping  
- **Pandas, NumPy** – Data cleaning & processing  
- **Tableau** – Data visualization  
- **Jupyter Notebook** (for data cleaning + analysis)

---
## 🔍 Key Insights
- Spam-related IPs were the most frequent type of abuse  
- Certain regions showed higher concentrations of malicious activity  
- Repeated patterns indicated possible coordinated attacks
