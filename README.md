# 🔍 Malicious IP Address Analysis

A project to collect, analyze, and visualize malicious IP data to uncover patterns in abuse types and regions.

---

## 🚀 What It Does

- Scrapes IP addresses and blocking reasons using Selenium  
- Classifies abuse types (e.g., spam, hijacking, proxy)  
- Cleans and structures the data  
- Visualizes insights in Tableau (country, type, IPv4 vs IPv6)

---

## 🛠 Tools Used

- **Python** (Selenium, Pandas)  
- **Tableau** (for dashboards)  
- **Jupyter Notebook** (for data cleaning + analysis)

---

## 📊 Key Visuals

- **Map**: Country-wise malicious IPs  
- **Bar/Pie Charts**: Abuse type breakdown  
- **IPv4 vs IPv6**: Comparison of threat volumes
